## Release notes for DiGidot Controller Software v3.3.1 - Beta 1 - Published on 26 of March ##

##### Added #####
* **LED IC: UCS7604**: Added native 16-bit mode (enable 16-bit input in the settings, to use 1:1 conversion).

##### Fixed #####
* **High: Backup & Restore**: In some cases, the 'Upload Backup file' was not showing on devices.
* **High: SD-card configuration**: Loading a configuration from an SD-card no longer breaks the IOconfig page.
* **Medium: Crossfade after trimming scene**: Performing a crossfade after trimming a scene (or finding a loop) now works correctly.
* **Medium: Generate (5-channel ICs)**: LED ICs with 5 channels now correctly respond when selecting the 4th or 5th color from the color picker.
